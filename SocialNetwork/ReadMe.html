Guide:

Der er seedet data til jer, som i kan manøvre rundt i.

Mange af manøvrene er lavet ved brug af hyperlinks

Applikationen er ikke så intuitiv, men i burde frem til det i ønsker med lidt god vile

Det er tænkt at alt det I skal tilgå er igennem Users og så arbejde sig frem derfra

I trykker på users og logger ind på den bruger i ønsker

Derefter kan i se forskellige oplysninger om brugeren

I kan gå ind på personens feed, circles eller wall

Derefter kan i se de tilknyttede posts

I kan desuden se/tilføje kommentare knyttet til posts

Fremgangsmåde:

generelt: *Users -> *Login -> *... )

feed: -> *Users -> *login -> *feed

wall: -> *Users -> *login -> *(Ønskede persons navn)

create post: -> *Users -> *login -> *(skriv post info) -> *ok

create comment: -> *Users -> login -> *(Feed/wall/circl/..) -> *post -> *(add comment)

Bemærk at hvis I opretter en circle så skal i kopiere CircleId og tilmelde brugeren når han er logget ind.
